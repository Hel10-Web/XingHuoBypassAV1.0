## XingHuoBypassAV1.0

图形化免杀🐎生成小工具，使用的python分离免杀+加密混淆，生成的结果放在result/exe目录下，ma.exe(名字可更改)和1.json文件放在目标机器同一目录下使用。

## 安装

需要pyinstall环境

```python
 pip3 install pyinstaller
 pip3 install -r requirements.txt
```
